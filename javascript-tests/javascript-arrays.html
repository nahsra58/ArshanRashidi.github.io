JavaScript Arrays

By: Arshan Rashidi

Numbers

When you click on the Show Numbers button, an event is fired that iterates through the array with a for each method. The array has been sorted lowest-highest.

Language Names

Similar to above, when you click the Language Name button, an event is fired that iterates through the array with a foreach method. The difference is the sort method alphabitizes the array without additional parameters.

Language Popularity

When you click the the Language Popularity button, an event is fired that iterates through the array of objects (popularLanguages). The objects have been pre-sorted by property (rank). Two properties are added through concatenation into one list item with formatting.
